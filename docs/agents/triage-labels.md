# Triage Labels

The Matt skills speak in terms of five canonical triage roles. This file maps
those roles to the label strings to use in this repo's issue tracker.

Needs confirmation: these default labels have not been confirmed against a live
tracker.

| Label in Matt skills | Label in this tracker | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Maintainer needs to evaluate this issue |
| `needs-info` | `needs-info` | Waiting on reporter for more information |
| `ready-for-agent` | `ready-for-agent` | Fully specified, ready for an agent |
| `ready-for-human` | `ready-for-human` | Requires human implementation |
| `wontfix` | `wontfix` | Will not be actioned |

When a skill mentions a role, use the corresponding label string from this
table.
