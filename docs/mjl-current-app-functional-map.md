# MJL Current App Functional Map

This file is current-state evidence only. It does not override
`docs/mjl-authoritative-decisions.md`.

## Executive Summary

The current application is a custom MJL workspace inside Dolibarr, not a
standalone application. Dolibarr provides authentication, users/groups/rights,
third parties, projects/tasks, ECM storage, and export support. MJL-specific
behavior is implemented in `custom/mjlfinancement`.

The module descriptor reports version `0.10.0`. Current code includes
production role/scope foundations, activity prevalidation/final validation,
expense prevalidation/final validation/disbursement, governed finance screens,
guarded document downloads, dashboards, alerts, reports/exports, invitations,
and audit helpers.

Legacy POC vocabulary remains in compatibility areas such as bootstrap scripts,
seed data, SQL backfills, route names, module descriptors, and some labels.
Those references are current-state/code debt, not target behavior.

## Current Workflow Evidence

### Activity

Current code includes:

- draft, submitted, correction requested, corrected, prevalidated, final
  validated, rejected, cancelled/completed-style statuses;
- `MjlActivity::prevalidate()`;
- `MjlActivity::finalValidate()`;
- no-self-review checks for creator/responsible users;
- activity execution fields and workflow audit entries.

Evidence:

- `custom/mjlfinancement/class/mjlactivity.class.php`
- `custom/mjlfinancement/activities.php`
- `custom/mjlfinancement/scripts/audit_schema_0.9.0.php`
- `custom/mjlfinancement/scripts/smoke_activity_workflow.php`

### Expense

Current code includes:

- requested, prevalidated, final validated, and disbursed amounts;
- submitted, prevalidated, final validated, disbursed, rejected/corrected
  workflow states;
- `MjlExpense::prevalidate()`;
- `MjlExpense::finalValidate()`;
- `MjlExpense::disburse()`;
- supporting-document and budget checks;
- no-self-review/disbursement checks.

Evidence:

- `custom/mjlfinancement/class/mjlexpense.class.php`
- `custom/mjlfinancement/expenses.php`
